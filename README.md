A complete, offline-capable personal finance management application that runs entirely in your browser. Track expenses, manage multiple accounts, create custom categories, and gain valuable insights - all with secure local storage.



## ✨ Features

### 📝 Transaction Management
- **Add, Edit, Delete** transactions with ease
- **Search and Filter** by type, category, or account
- **Bulk Import/Export** via CSV files
- **Real-time balance updates** across all accounts

### 📊 Analytics & Insights
- **Interactive Charts** - Visualize income vs expenses trends
- **Category Breakdown** - See where your money goes
- **Smart Insights** - AI-powered spending recommendations
- **Monthly Trends** - Track financial patterns over time

### 📁 Custom Categories
- **Create custom expense categories** (e.g., Coffee, Streaming, Hobbies)
- **Create custom income categories** (e.g., Freelance, Dividends, Bonuses)
- **Edit or delete categories** with automatic transaction updates
- **Organized by type** - Separate lists for expenses and income

### 🏦 Multi-Account Support
- **Create unlimited accounts** (Cash, Bank, Credit Card, etc.)
- **Set initial balances** - Without affecting income reports
- **Track account-specific balances** in real-time
- **Automatic balance calculation** from transactions

### 💾 Data Safety
- **IndexedDB Storage** - Secure, large-capacity local database
- **Backup & Restore** - Export/import complete database as JSON
- **CSV Export** - Download transactions for spreadsheet analysis
- **No cloud required** - Your data stays on your device

### 🎨 User Experience
- **Dark Mode Toggle** - Easy on eyes for night use
- **Responsive Design** - Works on desktop, tablet, and mobile
- **Toast Notifications** - Instant feedback for all actions
- **Keyboard Friendly** - Quick tab navigation
